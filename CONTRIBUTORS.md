# Contributors

> "We learn big things from small experiences" - **Bram Stoker**

Dracula Theme is an open-source project driven by and for the community.

## Recommendations

This theme follows the [Dracula Theme specification](https://draculatheme.com/spec) and uses the official [color palette](https://draculatheme.com/contribute#color-palette).

Before submitting changes, please format your code with:

```bash
npx prettier . --write
```
